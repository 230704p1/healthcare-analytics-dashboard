# healthcare-analytics-dashboard
Healthcare Capacity Analytics Dashboard using Streamlit and Python

## Project Overview

This project analyzes healthcare system capacity and care load for unaccompanied children using interactive data visualizations.

The dashboard provides insights into:

* Total Children Under Care
* Net Intake Pressure
* Care Load Volatility
* Backlog Accumulation
* Discharge Performance

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
* NumPy

## Features

* Interactive KPI Cards
* Date Range Filters
* Trend Analysis
* Correlation Heatmap
* Distribution Analysis
* Download Filtered Dataset
* Professional Dark Theme Dashboard

## Key Visualizations

1. Total System Load Trend
2. HHS Care Trend
3. CBP Custody Trend
4. Net Intake Trend
5. Monthly Discharge Analysis
6. Children Apprehended Analysis
7. HHS Care Distribution
8. Correlation Heatmap
